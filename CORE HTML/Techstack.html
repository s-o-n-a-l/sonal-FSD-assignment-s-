List of full stack development of tech stack

A stack is an arrangement of “things” kept in order one over the other. Tech stacks are sets of technologies that are stacked together to build any application. eg:web application stack.
Frontend tech stack
HTML (Hypertext Markup Language) Defines the basic layout and structure of your pages. For example, the sections,andtables are created using HTML tags.
CSS All the styling that's applied to the web pages comes from CSS — for example, the green theme of the MongoDB pages.
JavaScript The dynamic elements of the page come from the JavaScript code written by developers.
Backend tech stack
The backend tech stack (or server-side tech stack) consists of many components, like the database, server, framework, and operating system.
